La Estantería – Librería Online

Desarrollo de un sitio web funcional para una librería online. Proyecto académico de diseño web con desarrollo front-end en HTML, CSS y JavaScript. El trabajo se centró en la creación de un catálogo dinámico con buscador en tiempo real, filtros por género y carga automática de libros desde un array en JavaScript, priorizando la organización, la estructura del código y la experiencia de aprendizaje técnico dentro de un entorno académico. El resultado es una interfaz simple y funcional que refleja el proceso de crecimiento en el área de desarrollo web.

Cliente

Proyecto académico – Universidad ORT Uruguay

Materia: Programación Web

Duración

4 semanas

Mayo / Junio 2025

Herramientas

HTML, CSS, JavaScript

El Desafío

Problema identificado y solución propuesta

⚠️

Problema

Diseñar y desarrollar un sitio web que permitiera explorar, buscar y filtrar un catálogo de libros sin utilizar frameworks ni bases de datos externas. El desafío principal fue integrar funcionalidad dinámica y rendimiento fluido utilizando únicamente tecnologías front-end básicas.

✓

Solución

Se implementó un catálogo de libros cargado mediante un array de objetos en JavaScript, junto con funciones de búsqueda y filtrado en tiempo real. El foco estuvo en la lógica del código, la claridad estructural y la funcionalidad del sitio, cumpliendo con los objetivos académicos de la materia. El proyecto fue entregado en formato de código al docente evaluador.
